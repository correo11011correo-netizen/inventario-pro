# Historial de Cambios - StockPro

## [v4.0.0] - 2026-03-22 (Versión Actual)
### Añadido
- **Gestión Total:** Sistema completo de ABM (Alta, Baja, Modificación).
- **Interfaz UI:** Botones de edición y borrado en lista.
- **Contadores:** Visualización de total de items en stock.

## [v3.9.0] - 2026-03-22
### Añadido
- **Módulo Clientes:** Campos para Nombre y WhatsApp.
- **Integración API WhatsApp:** Generación automática de enlaces `wa.me` con mensaje formateado.

### Mejoras Críticas (Core)
- **Motor HD:** Habilitación de resolución 720p.
- **Algoritmo EAN-13:** Validación matemática de checksum para evitar lecturas erróneas.
- **Cooldown:** Sistema de enfriamiento de 2s tras lectura exitosa.
