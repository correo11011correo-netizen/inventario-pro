# 🚀 Inventario Pro v4.0 - POS Móvil HD

Sistema profesional de gestión de inventario y punto de venta (POS) optimizado para dispositivos Android. Funciona 100% en el navegador como una **PWA (Progressive Web App)** con precisión industrial.

## 🌟 Características Principales
- **Escáner de Alta Precisión:** Motor de lectura HD con validación matemática EAN-13 integrada.
- **Gestión de Stock:** Panel completo para agregar, editar y eliminar productos en tiempo real.
- **Presupuestos Instantáneos:** Calcula totales automáticamente mientras escaneas en modo venta.
- **Conexión Directa WhatsApp:** Abre chats con clientes (usando wa.me) y envía presupuestos formateados sin agendar el número.
- **Privacidad Total:** Almacenamiento local (localStorage). Los datos nunca salen de tu dispositivo.

## 🛠️ Tecnología de Precisión (¿Por qué es tan rápido?)
El sistema utiliza un algoritmo de **Validación de Checksum EAN-13**. Esto significa que cada lectura de la cámara es verificada mediante una ecuación matemática antes de ser aceptada, eliminando errores de lectura por movimiento o reflejos. Además, utiliza **aceleración de hardware** nativa de Android para decodificar a 30 FPS en resolución 720p.

## 📲 Instalación en Android
1. Abre el link en **Google Chrome**.
2. Toca los tres puntos (⋮) arriba a la derecha.
3. Selecciona **"Instalar aplicación"** o **"Añadir a pantalla de inicio"**.
