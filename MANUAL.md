# 📘 Manual de Usuario - StockPro v4.0

Bienvenido a StockPro, su sistema profesional de gestión de inventario y ventas.

## 🚀 Instalación (Android)
1. Abra el enlace provisto en **Google Chrome**.
2. Toque el menú de tres puntos (⋮) en la esquina superior derecha.
3. Seleccione **"Instalar aplicación"** o **"Añadir a pantalla de inicio"**.
4. La App aparecerá en su menú como una aplicación nativa.

## 📦 Gestión de Stock (Inventario)
Esta pestaña es para dar de alta sus productos.

1. **Escanear:** Apunte la cámara al código de barras.
   - *Tip:* Mantenga el teléfono a 15-20cm del producto.
   - El borde se pondrá **VERDE** cuando el código sea validado.
2. **Completar Datos:** Ingrese Nombre, Precio y Cantidad inicial.
3. **Guardar:** Pulse "GUARDAR / ACTUALIZAR".
4. **Editar/Borrar:** 
   - Toque un producto de la lista para editarlo.
   - Toque el icono 🗑️ para eliminarlo permanentemente.

## 🛒 Modo Ventas (Presupuestos)
Use esta pestaña cuando esté atendiendo a un cliente.

1. **Escanear para Agregar:** Cada escaneo suma 1 unidad al carrito.
2. **Ver Total:** El total se actualiza en tiempo real arriba.
3. **Enviar:** 
   - Ingrese el Nombre del Cliente.
   - Ingrese el WhatsApp (con código de área, ej: 549...).
   - Pulse **"📲 WhatsApp"** para generar el pedido y abrir el chat.

## ⚠️ Mantenimiento de Datos
**IMPORTANTE:** Sus datos viven en su teléfono.
- Use el botón **"EXPORTAR"** (arriba a la derecha) una vez por semana para guardar un respaldo en Excel.
- Si borra el historial de navegación de Chrome, podría perder sus datos. ¡Haga respaldos!
